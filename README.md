# GoodMoodMusicfestival Website
## Description
A Web for browsing festival lineup and registering for events.

## Tech Stack
- HTML , CSS , JavaScript
- Python (Flask)
- MySQL

## Feature 
- View Event lineup
- Register for events
- Store user data in Database
